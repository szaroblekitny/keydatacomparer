Keydatacomparer

Short description

This application is to fast compare the key data (such configuration parameters) between two databases (or two schemas in one database). As input parameter is a XML file with list of tables to compare, same in both databases. The program connects to databases, reads the list of tables to compare and their fields, compares data in records with the same data in primary key fields in compared tables, and finally produces report as a plain text file: records missing in databases and records with differences. It assumes that compared tables have primary keys and on it is based comparison.

Note

Currently only Oracle database (11g) is supported. PostgreSQL and MySQL are planned.

Usage

java -jar keydatacomparer-version.jar conf_file.xml [output_file.txt]

conf_file.xml must conform keydatacomparer.xsd schema, something like this:

<keydatacomparer>
 <databases>
  <sourcedatabase>
   <host>localhost</host>
   <port>1521</port>
   <name>WZDATA</name>
   <username>hr</username>
   <userpassword>password</userpassword>
  </sourcedatabase>

  <compareddatabase>
   <host>localhost</host>
   <port>1521</port>
   <name>WZDATA</name>
   <username>scott</username>
   <userpassword>password</userpassword>
  </compareddatabase>
 </databases>

 <tables>
  <table>KONFIGURACJA</table>
 </tables>
</keydatacomparer>

output_file.txt is name of file to produce filnal (plain text) report.
